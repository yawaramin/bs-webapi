Module Yawaramin_BsWebapi.OCamldoc

module Yawaramin_BsWebapi.OCamldoc: sig .. end
Used by the docs script to post-process generated HTML OCamldocs. Currently, linkifies all type and value names so that reader can click on them to get an anchor link to the specific name.

The output JavaScript is meant to be run in the docs subdirectory. See scripts/docs.sh for the actual usage.
