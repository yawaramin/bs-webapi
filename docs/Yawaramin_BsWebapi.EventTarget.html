Module Yawaramin_BsWebapi.EventTarget

module EventTarget: Yawaramin_BsWebapi_EventTarget
Interface implemented by objects that can receive events and may have listeners for them.

See https://developer.mozilla.org/en-US/docs/Web/API/EventTarget

module Common: Yawaramin_BsWebapi_Common
type intf = [ `eventTarget ] 
val make : unit -> intf Common.t
val addEventListener : typ:string ->
       listener:[ `eventHandler of 'a Common.eventHandler
                | `eventListener of 'a Common.eventListener ] ->
       ?options:[ `options of < capture : bool; once : bool; passive : bool > Js.t
                | `useCapture of bool ] ->
       unit
addEventListener ~typ ~listener ?options t. Register an event handler of a specific event type on the EventTarget. See https://developer.mozilla.org/en-US/docs/Web/API/EventTarget/addEventListener

listener must contain an event handler BuckleScript uncurried function.

removeEventListener ~typ ~listener ?options t. Removes an event listener from the EventTarget. See https://developer.mozilla.org/en-US/docs/Web/API/EventTarget/removeEventListener

listener must contain an event handler BuckleScript uncurried function.

val dispatchEvent : ([> intf ] as 'a) Common.t ->
       'a Common.event -> unit
Dispatches an Event at the specified EventTarget, invoking the affected EventListeners in the appropriate order. See https://developer.mozilla.org/en-US/docs/Web/API/EventTarget/dispatchEvent
